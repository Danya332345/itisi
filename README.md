# Task Manager – консольное приложение

**Автор:** Иван Иванов  
**Описание:**  
Консольный менеджер задач с поддержкой ООП, MVC, приоритетной очереди, стека отмены, фильтрации и JSON-сериализации. Позволяет добавлять, редактировать, удалять задачи, просматривать их по приоритету, фильтровать по статусу/приоритету, отменять последние действия и сохранять/загружать данные.

## Запуск
1. Убедитесь, что установлен Python 3.6+.
2. Скопируйте все файлы проекта в локальную папку.
3. Запустите из терминала:
   ```bash
   python main.py
